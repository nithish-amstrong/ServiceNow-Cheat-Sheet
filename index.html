General Best Practices
  • Client vs Server: Use Client Scripts/UI Policies for form behavior; Business Rules/Script Includes for backend logic.
  • Declarative vs Scripted: Start with no-code (e.g., Flow Designer), script if needed.
  • New vs Legacy: Prefer Flow Designer/IntegrationHub over Workflow/SOAP where possible.
  • Scoped vs Global: Use Scoped Apps to isolate changes and avoid conflicts.
  • Security First: Define ACLs, roles, and groups early in design.
